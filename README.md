# vue-form-design

基于 vue 和 element-ui 实现的可视化表单设计器

[ ![showUrl](https://img.shields.io/badge/%E6%BC%94%E7%A4%BA%E5%9C%B0%E5%9D%80-v0.1.0-orange.svg)](https://happydemoney.github.io/vue-form-design)

# 特性

* 可视化配置页面
* 一键预览配置的效果
* 一键生成配置json数据
* 一键生成代码，立即可运行
* 支持表单验证
* 快速获取表单数据

# 第三方插件

* vuedraggable
* element-ui
